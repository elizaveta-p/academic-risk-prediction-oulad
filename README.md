# Прогнозирование академического риска студентов

Pet-project по предсказанию студентов группы риска на основе данных об обучении, результатах оценивания и активности в LMS на датасете OULAD.

## Цель проекта
Построить модель бинарной классификации, которая определяет, находится ли студент в группе академического риска.

## Что сделано
- объединены несколько таблиц датасета OULAD
- выполнен первичный анализ данных (EDA)
- сформированы признаки на основе:
  - результатов оценивания
  - активности в LMS
  - регистрационной информации
- создана целевая переменная `at_risk`
- выполнено разбиение данных с учетом студентов (без утечки)
- обучены и сравнены модели:
  - Logistic Regression
  - Random Forest
- проведена оценка качества моделей:
  - Accuracy
  - Precision / Recall / F1-score
  - ROC-AUC
  - Confusion Matrix
- выполнен анализ важности признаков
